# Eventra Frontend

Eventra Frontend is the interface of an event management web application where users can discover events, explore event details, manage their profile and access organizer-related pages through a modern and elegant design.

This project represents the frontend stage of Eventra and focuses on structure, layout and user experience.

## About the Project

Eventra is a web platform created for event discovery and event management. It is designed for two types of users:
- guests, who can browse and join events
- organizers, who can create and manage their own events

The frontend version presents the visual concept of the platform and its main user flow.

## Built With

- HTML5
- CSS3

## Pages Included

- Home
- Events
- Event Details
- Create Event
- Profile
- Edit Profile
- Sign In
- Register
- About Us

## Features

- modern landing page with featured events
- event listing and event details pages
- profile page with QR code section
- organizer event creation form
- sign in and registration pages
- waiting list and notifications interface
- clear navigation between pages
- clean and consistent design

## Purpose

The goal of this project was to design the frontend of a complete event platform with a user-friendly interface and a clear page structure.

## Status

This repository contains the frontend version of Eventra.
