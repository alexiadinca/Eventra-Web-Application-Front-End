About Eventra

Eventra is a modern web platform designed to help users discover events, connect with organizers, and explore experiences happening around them. The platform allows guests to browse events and organizers to publish conferences, exhibitions, openings, tastings and many other types of events.

University Project

Eventra was developed as a university project for the course Web Applications Design.

The goal of the project is to design and implement a complete event management platform where users can create accounts, browse events, register for experiences and organizers can publish and manage their own events.

This project demonstrates skills in:

  • HTML5 and semantic web structure
  • CSS layout and responsive design
  • JavaScript front-end logic
  • UI/UX design principles
  • Web application architecture
